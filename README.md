# Arctic Quest

Ezen állomány a PENdroid 2023 verseny harmadik fordulójára készített, **Arctic Quest** névre keresztelt játék forráskódját tartalmazza.
A játék két fő összetevőből áll: egy csúszási fázisból, amelyben a pingvinek egy jeges hegyen csúsznak lefele, majd egy csatafázisból, amikor a hegy lábánál gonosz lényekkel találkoznak össze. A játék során pingvineket oldasz fel, melyeket fejlesztési pontok árán fejleszthetsz. A játék előrehaladtával egyre erősebb ellenfelekkel és nehezebb pályákkal találod magad szembe, de a jutalmak is nagyobbak!
